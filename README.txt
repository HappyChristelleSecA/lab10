H's Phone Case Gallery - Single Page Website

Project Overview
This project is a single-page website for H's Phone Case Gallery, showcasing a collection of stylish phone cases. Built with HTML5, CSS3, and Bootstrap 5, it demonstrates responsive web design principles and modern front-end practices.

Features
- Responsive single-page design
- Navigation menu with smooth scrolling
- Home, About, Gallery, and Contact sections
- Image gallery of phone cases
- Contact form
- Social media links in footer

Technologies Used
- HTML5
- CSS
- Bootstrap 5.3.0
- Font Awesome 6.4.0

Viewing Instructions
1. Download the HTML file.
2. Open the file in any modern web browser (Chrome, Firefox, Safari, or Edge).

Testing Guide
1. Responsive Design:
   - Resize browser window to test responsiveness.
   - Use browser dev tools to emulate different devices.

2. Navigation:
   - Click menu items to test smooth scrolling to sections.
   - Check hamburger menu functionality on small screens.

3. Gallery:
   - Verify all images load correctly.
   - Check gallery layout on different screen sizes.

4. Contact Form:
   - Test form validation by submitting without filling fields.
   - Ensure form is readable on all screen sizes.

5. Cross-browser Testing:
   - View the page in different browsers to ensure consistency.

Development Notes
- All content is in a single HTML file for simplicity.
- Styles are included in the `<style>` tag within the HTML.
- Replace placeholder images with actual product photos.
- Update social media links in the footer.

Future Enhancements
- Add backend functionality to the contact form.
- Implement a shopping cart system.
- Create individual product pages.
